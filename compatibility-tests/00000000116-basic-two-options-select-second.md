# Basic Two Options Select Second

Simple choice between two options, selecting the second option.

## Script
```cuentitos
What do you want to do?
  * Go left
    You went left
  * Go right
    You went right
```

## Input
```input
2
s
```

## Result
```result
START
What do you want to do?
  1. Go left
  2. Go right
> Selected: Go right
You went right
END
```
